# ATLAS-2 Demo Checklist

## Pre-Demo
- Start dev server: `npm run dev`
- Open: `https://localhost:5174/` (accept the local cert)
- Optional: confirm Quest access at `https://<your-pc-ip>:5174/`

## Mnemosyne (Key 3)
- Nodes and cluster labels visible.
- Green AI edge handle pulses.
- Select handle -> trust panel opens.
- Accept turns edge blue; Reject hides edge/handle.

## Themis (Key 4)
- Places, transitions, arcs visible.
- Token moves along the path.
- Checkpoint panel appears and pauses flow.
- Continue resumes movement to the next place.

## Tecton (Key 5)
- Base module renders.
- Prev/Next buttons change size and stats.
- AI toggle shows side-by-side ghost module.

## Controls
- `H`: toggle hint overlay
- `D`: toggle debug overlays
- VR: use the floating Module Menu panel to switch demos
